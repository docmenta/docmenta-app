User Interface Settings

User Interface Skin
The selected skin defines the overall UI design, as for example UI colors, borders, button design and so on. By default only one skin is installed, which is selected by default. However, additional skins may by installed through CKEditor plufg-ins. This option reflects the setting CKEDITOR.config.skin in the CKEditor API.
User Interface Color
The base user interface color to be used by the editor. Not all skins are compatible with this setting. This option reflects the setting CKEDITOR.config.uiColor in the CKEditor API.
Dialog Background Opacity
As long as a modal dialog is opened, the parent window is disabled, which is indicated by greying out the parent window. The entered value defines the opacity used for background cover. This option reflects the setting CKEDITOR.config.dialog_backgroundCoverOpacity in the CKEditor API.
Toolbar Location
The position in the editor window where the toolbar will be displayed. This option reflects the setting CKEDITOR.config.toolbarLocation in the CKEditor API.
Remove Buttons
A list of toolbar button names that shall be hidden. This will also work for non-button toolbar items, like the Font drop-down list. This option reflects the setting CKEDITOR.config.removeButtons in the CKEditor API.
Example: Underline,JustifyCenter
