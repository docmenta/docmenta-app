Editor Options

Paste as plain text
If this option is checked, then all pasting operations are forced to insert plain text into the editor, losing any formatting information possibly available in the source text. This option reflects the setting CKEDITOR.config.forcePasteAsPlainText in the CKEditor API.
Ignore empty paragraph
Whether the editor must output an empty value (''), if its content only consists of an empty paragraph. This option reflects the setting CKEDITOR.config.ignoreEmptyParagraph in the CKEditor API.
Alternative text required for captioned image
Determines whether alternative text is required for captioned images. This option reflects the setting CKEDITOR.config.image2_altRequired in the CKEditor API.
Disable image resizer
If this option is checked, disables the image resize functionality. This option reflects the setting CKEDITOR.config.image2_disableResizer in the CKEditor API.
Disable built-in browser spell check
If this option is checked, disables the built-in spell checker (in case the browser provides one). This option reflects the setting CKEDITOR.config.disableNativeSpellChecker in the CKEditor API.
Disable native table editing (Firefox)
If this option is checked, disables the table tools offered natively by the browser (currently Firefox only) to perform quick table editing operations, like adding or deleting rows and columns. This option reflects the setting CKEDITOR.config.disableNativeTableHandles in the CKEditor API.
Confirm cancel in dialog window
If this option is checked, the user is asked to confirm close (cancel), if any dialog field was modified. This option reflects the setting CKEDITOR.config.dialog_noConfirmCancel in the CKEditor API.
