2.3.8.9. Quick-Links

Instead of manually creating a link as described in Section 2.3.8.8, “Links”, you can enable Quick-Links to allow faster creation of links. If the Quick-Links option is enabled you can just type the text [[node_alias]] to create a link to the section identified by alias node_alias. When saving the content, the pattern [[...]] is automatically transformed to a link.
See Section 2.7.2.2, “Editing the user profile” on how to enable Quick-Links. For an example on how to use Quick-Links see Section 1.4.1, “Referencing sections”. Following table gives an overview of the supported Quick-Link patterns that can be used to create different types of links:
PatternDescription
[[#alias]]
[[alias]]
 
Creates a link to the content node identified by alias. The # character preceding the alias name is optional. The title of the referenced node is used as link text.
Example:
[[#intro]]
Creates a link to the node identified by alias name intro.
[[alias link text]]
Creates a link to the content node identified by alias. The text following the alias name is used as link text.
Example:
[[intro see here]]
Creates a link to the node identified by alias name intro using the link text "see here".
[[alias | alternative text]] 
Creates a link to the content node identified by alias. The text following the '|' character is used as alternative link text. The alternative link text is only displayed, if the node identified by alias does not exist or is not included in the exported publication (or in any of the referenced publications; see Section 2.6.1, “Publication configuration”). Otherwise the title of the referenced node is used as link text.
Example:
[[#intro | Introduction]]
Creates a link to the node identified by alias name intro. If the referenced node does not exist (within the exported publication or in any referenced publication), then the link is replaced by the fixed text "Introduction".
[[file/alias]]
[[file/alias link text]][[file/alias | alternative text]]
Creates a link to the file node identified by alias. The text following the alias name is used as link text. If no link text is provided, then the name of the referenced file is used as link text. If the link text is preceded by a '|' character, then the link text is only displayed, if the node identified by alias does not exist or is not included in the exported publication (or in any of the referenced publications; see Section 2.6.1, “Publication configuration”).
[[image/alias]]
[[image/alias link text]][[image/alias | alternative text]] 
 Creates a link to the image identified by alias. The text following the alias name is used as link text. If no link text is provided, then the image title is used as link text. If the link text is preceded by a '|' character, then the link text is only displayed, if the image identified by alias does not exist or is not included in the exported publication (or in any of the referenced publications; see Section 2.6.1, “Publication configuration”).
[[url]]
[[url link text]]
 
 Creates an external link to the given URL. The text following the URL is used as link text. If no link text is provided, then URL is used as link text.
Example:
[[http://www.docmenta.org Homepage]]
