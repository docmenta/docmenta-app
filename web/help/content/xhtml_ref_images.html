3.6.1. Inline Images
3.6.2. Figures
Following table shows the supported attributes of the img element, which can be used to insert images into the content. Note that other attributes may work as well, but only the attributes listed below are known to be supported for interactive and print output.
ElementSupported
attributes		Comment
img
id, class, height, src, style, title, width
Defines an inline image or figure. For details see the following sub-sections.
